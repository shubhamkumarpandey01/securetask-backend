# SecureTask Backend API

Secure REST API with JWT authentication, role-based access, validation, and task CRUD operations.

## Tech Stack

Node.js, Express.js, MongoDB, Mongoose, JWT, bcryptjs, express-validator

## Features

- User registration and login
- Password hashing
- JWT authentication
- Protected routes
- Create task
- Get tasks
- Update task
- Delete task
- Input validation
- API versioning using /api/v1

## Run Locally

```bash
npm install
npm run dev
